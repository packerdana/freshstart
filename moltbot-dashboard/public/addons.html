Recommended Add-ons

1) Web Search (Brave)

Adds research + Reddit/NALC digging with sources. Requires a Brave Search API key.
Steps:
1) Get a Brave Search API key
2) In Control UI → Config, set:
   tools.web.search.enabled = true
   tools.web.search.provider = "brave"
   tools.web.search.apiKey = "..."
3) Restart gateway

(If you want, I can add a config form here next.)

2) Browser Relay / Extension

Lets Moltbot drive a real browser tab for Vercel/Supabase/GitHub admin tasks.
Steps:
1) Install Moltbot Browser Relay extension
2) Click the extension on the tab you want to “attach”
3) Use the Control UI Chat to ask Moltbot to operate that tab

Note: best for "take the wheel" tasks.

3) Remote access (Tailscale Serve)

Safely access Control UI when you’re away from the laptop.
Steps:
1) Install + login to Tailscale
2) Run:
   moltbot gateway --tailscale serve
3) Open your MagicDNS URL

This keeps your gateway bound to loopback and uses HTTPS.

4) Better Memory (optional)

Better long-term recall (bugs fixed, decisions, prefs). Requires API key depending on provider.
Option: memory-lancedb plugin
- Pros: auto-capture/recall
- Cons: needs an embeddings key and uses more resources

We can keep it off until you want it.
